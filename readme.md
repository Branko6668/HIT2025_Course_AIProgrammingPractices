## 🧠 目录

### ✅ No1_nQueens：N 皇后问题求解器
- 基于回溯算法实现任意规模的皇后摆放
- 图形化展示解法过程与摆放效果
- 支持用户输入 `n` 并实时显示结果

### ✅ No2_KMeans_KNN：可视化数据分析平台
- 聚类分析：KMeans + Elbow Method（肘部法）图形辅助选择 `k`
- 分类识别：KNN 分类器 + 性能指标分析
- 支持 Excel/CSV 数据加载，界面交互友好
- 内置 Q&A 学习提示，便于理解算法原理



## 🛠 使用说明

运行方式（以任意模块为例）：

```bash
cd No2_KMeans_KNN
python Kmeans_KNN.py
```
